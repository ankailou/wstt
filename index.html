What is Web Scraping?

Web scraping is the act of retrieving data and other information from online webpages. There are many different ways to do this; From copy-pasting the data by hand, to requesting a professional web crawling comanpy to retrieve that data you need, to writing short scripts in your favorite language. For as many ways as there are to retrieve the data, there is just as many reasons as to why you would want to web scrape. Some applications of web scraping include: DBPedia- a free query-able database of wikipedia infoboxes, Price Comparison applications, Website Archiving, News Aggregation applications, and many more!

Issues Concerning Web Scraping

While web scraping is widely used and provides an efficient way to collect important data, it resides in a somewhat grey area legally. It is important to do the following before scraping a website:

  • Check the website's terms of use
  • Do not scrape copyrighted content
  • Try not to burden the websites servers and services
  • Avoid scraping sensitive user information

Following these guidelines will help to ensure that any web scraping application you make stays on the ethical high ground. Another reason that web scraping is controversial is because of how applications can compete with the website that is scraped. One example of this is a ticket pricing platform that aggregates all ticket prices from competing airlines. This platform compares the prices and allows buying of tickets right through their platform. This circumvents the airlines' websites and takes away from their user numbers. It is important to keep the purpose of your application in mind and make sure to consider how it would affect the sites you scrape from.

Another important aspect of web scraping is the fact that it can be automated to an extent that it can greatly hinder websites' servers. This automation has lead to large numbers of bots scraping information from many different sources. In order to combat the wide-spread use of web scraping bots some websites have set up countermeasures that include: Blocking IPs, Traffic monitoring, Captcha requirements, and even data obfuscation. When attempting to scrape information for yourself you may encounter some of these web scraping bot countermeasures.
